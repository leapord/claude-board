📐 设计说明（实现时参考）

数据模型（settings.configuredProjects）：

[
  { id: 'p_xxx', name: 'claude-board', path: '/Users/.../claude-board',
    addedAt: '2026-05-12T...', lastOpenedAt: '2026-06-06T...' }
]

"启动 Claude" 按钮行为（main 进程 IPC projects:launch）：

  • macOS：spawn osascript -e 'tell app "Terminal" to do script "cd \<path\> && claude"'（新窗口）
  • Windows：spawn cmd /c start "Claude" cmd /k "cd /d \<path\> && claude"
  • 同时调 cb.projects.touch(id) 更新 lastOpenedAt

与概览屏联动：

  • 概览"最近项目"卡片 = configuredProjects 按 lastOpenedAt 倒序前 4 个
  • 概览"最近活动" = configuredProjects.filter(p => p.lastOpenedAt > 7天前)
  • 如果 configuredProjects.length === 0，两区都显示空态 + 引导去屏 2

scanner 行为变更：

  • 不再自动遍历 ~/.claude/projects/ 根目录聚合项目
  • 只聚合 session.cwd 匹配某个 configuredProject.path（或子目录）的会话
  • 如果没配置项目 → 返回空数据（不用 seedSynthetic 兜底）
